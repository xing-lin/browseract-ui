## 注意事项

1. 图标默认作为无语义的装饰性图标 `aria-hidden="true"` ，没有用 `React.forwardRef` 来传入 `ref`, 如果需要点击事件，需要用 `<button>` 包裹，如果不包含其他文本 button 要添加 `aria-label`。

   ```tsx
   // Good
   <button
     type="button"
     aria-label="Copy"
     onClick={onClick}
   >
     <IconCopy />
   </button>

   // Bad
   <IconCopy onClick={onClick} />
   ```

2. Icon 中的大小跟颜色，**可通过设置父级元素来传递**, `font-size` 和 `color`。
3. 双色 Icon 目前是黑色 + 绿色，hover 可添加 `._icon-hover-duo-color` 达到悬浮时改色的效果。

   ```tsx
   <button className="_icon-hover-duo-color">
     <IconDuoAgent {...args} />
   </button>
   ```
